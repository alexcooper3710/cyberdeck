# Phase 6 — Cyberdeck Integration

**Goal:** Package the working carrier into a physical cyberdeck enclosure.

## Tasks

- [ ] Mechanical design (case dimensions, mounting points)
- [ ] Integrate laptop display via LCD controller board (~$25)

- [ ] USB-C PD trigger + laptop battery integration
- [ ] PWM fan (controlled by FPGA pin)
- [ ] Front panel cutouts for D1/D2/D3 + reset/boot buttons + cart slot access
- [ ] Audio via I2S DAC + speakers

## Cost estimate

$50–200 for enclosure (3D printed cheap, sheet metal more)
